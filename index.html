Attributes

Attribute Type Default Description
variant string "solid" Defines the visual style of the button
size string "md" Sets the size: sm, md, lg
disabled boolean false Disables the button
icon string null Adds an icon by name (if supported)
type string "button" Sets the native button type: button, submit, etc.

What is FujinKit?

FujinKit is a UI components library based on Web Components. Every UI library out there aims to be elegant, pixel-perfect, and full of life.

We do not.

We exist to serve the SASS (the traditional kind) builders of the world. Our mission? Create boring, lifeless, but highly usable components.
